# Laravel-API
API para gerenciar contatos
  
- Colocar o banco mysql (api_bd.sql)
- Importar no Postman o colletion.json (Api Laravel - 08-2019.postman_collection.json)
- Versão do Laravel utilizada foi a 5.8.35
